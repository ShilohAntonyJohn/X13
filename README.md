X13 Polish Language learning app is written in Rust with Bevy game emgine and teaches Polish Grammar Cases through falling consonants where noun/adjective has to catch them and it also teaches Polish letters. V13 which is written in Python gives 4.5k+ Polish words with its pronounciation generated by AI with high accuracy. It porvides mnemonic combinations and imaginative sentences for each mnemonic combination for easy visaulization leading to mastery of the language!

For Linux Users:

Download  X13_and_V13_Linux from github releases https://github.com/ShilohAntonyJohn/X13/releases

To run V13

For Fedora:

$ sudo dnf install python3-tkinter python3-devel

$ pip install --upgrade pip

$ pip install pygame customtkinter

Navigate to unzipped release

run:

$ python V13.py

For X13

$ ./X13

For Windows

Install python, pygame and customtkinter

run  python V13.py

For X13

Click on X13.exe
